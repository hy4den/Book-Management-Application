# Kitap Yönetim Uygulaması

Bu proje, Android Studio ve Java kullanılarak geliştirilmiş bir kitap yönetim uygulamasıdır. SQLite veritabanı ile kitap bilgileri saklanır ve yönetilir.

## Özellikler
- Kitap ekleme
- Kitap silme (tekli/toplu)
- Kitap güncelleme
- Kitap sorgulama
- Kitap listesi görüntüleme

## Kullanılan Teknolojiler
- Java
- Android Studio
- SQLite
- Gradle

## Kurulum
1. Proje dosyalarını indirin veya kopyalayın.
2. Android Studio ile açın.
3. Gerekirse Gradle senkronizasyonunu başlatın.
4. Uygulamayı emülatörde veya gerçek cihazda çalıştırın.

## OOP Prensipleri
- Her işlev için ayrı sınıflar
- Veri gizleme, kalıtım ve çok biçimlilik

## Kitap Veri Modeli
- ID (benzersiz)
- Kitap Adı
- Yayın Yılı
- Yazar
